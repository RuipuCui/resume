# Resume Builder

This project is built with React, Vite, TypeScript and Tailwind CSS.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the URL shown (usually http://localhost:5173).

4. To save as PDF:
   - Open the page in your browser.
   - Press `Cmd + P` (Mac) or `Ctrl + P` (Windows).
   - Choose "Save as PDF".
   - Ensure "Background graphics" is checked in print settings to include the blue header.
   - Set margins to "None" or "Default" as needed (CSS handles most).
